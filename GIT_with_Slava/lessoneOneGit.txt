   1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано 
предварительно?

Для того что бы скопировать созданный репозиторий на GIThub на свой компьюте, его сначала нужно создать на GIThub. Для этого нужно в правом 
верхнем углу выбрать new repositiry и создать его , затем через зеленую ссылку code скопировать ее и на своем компьютере через команду git 
clone добавить ссылку и нажать ввод.
